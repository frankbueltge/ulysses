# Research Foundation — five tranches

This directory carries the complete five-tranche Research Foundation from which
Protocol v4 was derived. `docs/RESEARCH-FOUNDATION-V1.md` remains the concise
anchor; these dossiers are the normative interpretation source behind it
(brought into this repository by amendment 2026-07-19 — before that they were
research records outside the repository).

Canonical origin: the Ulysses v4 protocol package v1.1 in
`frankbueltge/frankbueltge.de` under `docs/ulysses-v4-protocol-package-v1.1/foundation-reference/`.
The copies here are verbatim; only the directory names are shortened
(`ulysses-research-foundation-tranche-N` → `tranche-N`).

## How this directory is used

Selective consultation, never total loading. The Foundation itself sets the
rule (tranche 1, interim implications §4B): *each project selects only the
concepts, cases and materials relevant to its situation; no total theoretical
context is loaded by default.* A run consults a dossier when a score invokes a
position the Foundation has mapped, when a §5.4 compose test needs its full
criteria, or when a strategy from the toolbox is considered — and cites the
dossier instead of reconstructing theory from model memory.

## Orientation

| Tranche | Focus | Entry points |
|---|---|---|
| `tranche-1` | artistic research: research about/for/through art | `03` concept dossier, `04` interim implications |
| `tranche-2` | knowledge, non-knowledge, exposition, method and rigour | `07` interim model (coupled exposition) |
| `tranche-3` | experimental systems, epistemic things, trace, difference/variation | `11` interim model (controlled underdetermination), `10` toolbox v0.1 |
| `tranche-4` | difference, repetition, event, trace, cartography | `11` integrated model (differential consistency §6), `10` toolbox v0.2 |
| `tranche-5-final` | agency, authorship, apparatus, infrastructure, correctives | `11` final synthesis, `12` binding requirements (FR-01…62), `10` **toolbox v0.3**, `13` readiness decision |

The toolbox referenced by Protocol §11 is
`tranche-5-final/10-TOOLBOX-CANDIDATES-V0.3.md` — 26 methodological
hypotheses, each with use conditions and a failure test. It is provisional and
revised by works, never a compulsory menu.

These files are historical research records. They are not edited here;
corrections to the Foundation happen as dated, human-approved decisions in the
canonical package, then mirrored.
