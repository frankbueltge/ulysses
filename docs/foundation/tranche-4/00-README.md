# Ulysses Research Foundation — Tranche 4
## Difference, event, trace, iterability and cartography

**Status:** Research foundation, not yet an active Ulysses protocol  
**Date:** 18 July 2026  
**Builds on:** Tranches 1–3

## Purpose

Tranche 4 addresses the main gap left by the experimental model developed in Tranche 3. That model gave Ulysses a credible experimental epistemology — controlled underdetermination, trace preservation, adversarial exposure and explicit limits — but not yet a sufficiently developed account of aesthetic form, event, repetition and cartographic action.

The guiding question is:

> How can an artistic research apparatus produce and hold open a consequential difference without reducing it to novelty, error, network imagery or a concept already known in advance?

## Principal correction

The tranche rejects four common equivalences:

```text
unexpected output != event
repeated execution != repetition
recorded metadata != constitutive trace
network visualisation != cartography
```

These distinctions matter directly for Ulysses. A language model can generate surprising outputs, a scheduler can run a process repeatedly, a repository can record every change, and an interface can display a graph. None of those operations is sufficient to establish an event, a productive repetition, an epistemically active trace or a map.

## Main result

The strongest integrated model is now:

> **Ulysses is a human-accountable, machine-operative apparatus of situated differential individuation. It constructs concrete problems, exposes them to encounters and iterable contexts, preserves constitutive traces, and composes local forms and maps through which changes in object, apparatus and possible action become perceptible and contestable.**

This formulation does not discard the Tranche 3 model of controlled underdetermination. It places it inside a stronger theory of how differences are generated, how events are claimed, how traces alter meaning and how form can hold heterogeneous relations together without resolving them into identity.

## Readiness decision

The Foundation is now conceptually strong enough to reject weak Ulysses proposals and to construct serious pilot scores. It is **not yet complete enough to freeze a final protocol**.

One final essential tranche is required:

> **Machine agency, authorship, infrastructure, situated responsibility and decolonial/feminist correctives.**

Without it, the model would still risk romanticising machine agency, treating its European philosophical corpus as universal and under-describing the labour, datasets, platforms and power relations that make Ulysses possible.

## Immediate practical consequence

The current nightly production routine should not continue merely while the Foundation is being completed. No final v4 runtime should be implemented from this tranche alone. However, the concepts and failure tests here are mature enough to evaluate proposed works and to construct one deliberately bounded pilot after the next tranche.
