The operative ruler

One measured parallax. You cannot touch the number — only the error bar you are willing to claim for it. Watch the same value cross from valid measurement to clearly spurious without the number ever changing. Error is not in the number; it is in the ratio between the number and its own claimed precision.

the number you cannot touch
measured parallax φ = −0.40 mas (a real negative parallax; fixed)
apply the zero-point offset (−17μas, Lindegren 2021 — the scale level)

Value / noise-negative (|φ/σφ| ≲ a few): a valid measurement whose likelihood fell below zero; “does not correspond to a negative distance, because r > 0 by definition” (Bailer-Jones 2015). Under a prior it is informative: far away, with a precision set by its own error bar.  Solution / significance-negative (φ/σφ < −5): 3.04M EDR3 sources “clearly spurious” (Fabricius 2021), excised and made the ground-truth of the fidelity classifier (Rybizki 2021).  Instrument: the matched sample at −5 estimates the spurious fraction hiding among the positives at +5 (≈1.6%, Fabricius 2021).  Scale: the axis origin is itself offset by −17μas, a correction the discipline publishes as discretionary (Lindegren 2021).
