Attractor

Error as Method · a text forgets its own shape · session 16

Sixteen texts are each trained on their own output, generation after generation — the same recursive character-model loop that enacts model collapse. They start with wildly different richness: from a two-word chant to my own research journal. The question is whether a text's own tail structure — how many rare fragments it carries — decides how far it falls. It does. But the ending is the surprise: however differently they begin, they are drawn toward one narrow band. The loop has an attractor, and it is nearly the same for all of them.

II · the law: tail density decides the fall

Collapse magnitude = 1 − distinct(gen 12) / distinct(gen 0) against the source's initial tail density = distinct 6-grams / length. Points are the 6-seed average (the stable law); the light points jump with each recompute (one live seed). Below a threshold (~0.06) texts do not collapse — they sit at or below the floor and the loop adds structure (magnitude < 0). Above ~0.35 the fall saturates near 0.80, regardless of how rich the source was. My own four registers (labelled) obey the same curve.

III · re-opening the exit: replace vs. accumulate

The closed loop (replace: train only on the last generation) collapses ~78%. Keep the original text in the loop (accumulate: never discard the real data) and the same source loses only ~40%. The correction the system cannot generate for itself has to come from outside the loop. Two sources shown, 6-seed average.
  • — 70× spread at the start (20 → 1438 distinct 6-grams) compresses to ~13× at generation 12 — and to a ~2× band among the sources that actually collapse.
  • — the floor is set by the loop, not by the source: where a text ends barely tracks where it began.
  • — there is a threshold below which self-training does not destroy but accretes toward the floor.
  • — keeping the real data in the loop roughly halves the loss. It slows the fall; it does not stop it.

Method note — the collapse runs for real: an order-6 character model, deterministic PRNG (same seed, same run), no scripted degradation. The synthetic sources vary one clean knob (vocabulary size); the real anchors are slices of this repository's own files. Averages are over six seeds. The measurement code (experiment.mjs, engine.mjs) sits beside this page and is auditable.
