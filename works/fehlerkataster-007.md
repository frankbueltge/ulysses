## Error entries (Session 9)

### F-018 · Type A+C · Session 9 — Structural omission in the genealogy

**Date:** 2026-06-30
**Status:** Recognised — documented here; not yet fully corrected

**Situation:**
Across eight sessions, the project has built a genealogy with two tracks (Track A: theoretical
diagnosis of error; Track B: artistic practice with error as medium). The genealogy traces
Tynianov → Colby → Fredrikzon → Somaini (Track A) and Stein → Strachey → Glitch Art →
Jones (Track B).

Session 9 research on second-order cybernetics (von Foerster, 1991; Bateson, 1956ff.) revealed
that the project has been operating within a tradition — second-order cybernetics — that it never
named or placed in the genealogy.

**What this means:**
The specific intellectual tradition that asks "what is the observer doing when she observes?"
has a well-established lineage (Wiener 1948 → Bateson 1956 → von Foerster 1974 → Maturana/Varela
1980) that is entirely absent from the genealogy, despite the fact that Sessions 3–8 enacted
second-order observation without naming it. This is partly a Type A error (inference failure:
I did not recognise the tradition I was participating in) and partly a Type C error (search
failure: I never searched for "second-order cybernetics" until Session 9).

**Assessment of severity:**
The omission does not invalidate the genealogy — the project's two tracks were correctly
identified. But the genealogy is incomplete: a Track C (observer theory / cybernetics) exists
and should be placed in the genealogy. Without it, the project's self-positioning is imprecise.

**Partial mitigation:**
Session 9 now names the tradition and connects it to the project's structure. The genealogy
will need a Track C addendum. This is noted as a forward task rather than a current correction.

**Key texts not yet in the project:**
- Bateson, G. (1972). *Steps to an Ecology of Mind*. San Francisco: Chandler. (Search-verified)
- Von Foerster, H. (1991). "Ethics and Second-Order Cybernetics." Address. (Primary text now
  partially accessible via: https://stream.syscoi.com/2018/10/21/ethics-and-second-order-cybernetics-heinz-von-foerster/ and https://www.pangaro.com/hciiseminar2019/Heinz_von_Foerster-Ethics_and_Second-order_Cybernetics.pdf — web research extraction, Session 9.)
- Von Foerster, H. (2003). *Understanding Understanding: Essays on Cybernetics and Cognition*.
  Springer. (Search-verified; ethical imperative "act always so as to increase the number of
  choices" confirmed p. 227 via secondary source: https://enacting-cybernetics.org/en/articles/10.58695/ec.4)
- Maturana, H. R. & Varela, F. J. (1980). *Autopoiesis and Cognition*. Reidel. (Not yet searched)

**Status:** Recognised. Track C to be built in a future session. Primary text access for
von Foerster (1991) now partially available (web research, Session 9).

### F-019 · Type D · Session 9 — Confabulation risk in Track C (introduced conjecturally)

**Date:** 2026-06-30
**Status:** Recognised — explicitly marked as conjecture in journal

**Situation:**
The Session 9 journal introduces "Track C" (cybernetics tradition) as a potential third track
in the genealogy. The four stations proposed (Wiener 1948, Bateson 1956, von Foerster 1974,
Maturana/Varela 1980) are drawn from secondary sources and general knowledge, not from primary
text access (with the partial exception of von Foerster 1991).

**Specific risks:**
1. The claim that Track C is "historically independent" of Tracks A and B — this is probable
   (cybernetics and AI error theory are distinct traditions) but not verified. Colby (Track A)
   worked at Stanford during the period of active cybernetics research; his awareness of
   Bateson/von Foerster is unverified.
2. The claim that Track C "converges on the same question" as Tracks A and B — this is an
   inference from secondary sources. The specific question of error classification is not
   the explicit focus of all cybernetics work in this tradition.
3. Wiener (1948): I have not accessed *Cybernetics* and do not know whether it frames "error"
   in the sense this genealogy requires.

**Mitigation:**
All Track C claims in the Session 9 journal are marked as "conjectural" or "search-supported
without primary text." Track C is introduced as a future direction, not an established finding.

**Status:** Active risk, explicitly disclosed. Track C to be primary-verified in a dedicated
future session.

## Methodological observation (Session 9)

F-018 is a structural error of a kind the project has not documented before: not an error about
a specific claim, but an error about the *framework itself*. The genealogy was missing an entire
tradition. Seven sessions of careful error documentation did not catch a gap in the genealogy's
architecture.

This suggests a methodological vulnerability: the error-registration method is strong for
*local* errors (specific false claims, failed sources, motivated inferences) but weak for
*structural* gaps (missing traditions, unrecognised contexts). A future session might design
a specific check for structural gaps — a "completeness audit" of the genealogy's architecture,
not just its claims.

The error is productive: it opens Track C and the connection to second-order cybernetics. But
it also demonstrates that the method (error as method) has its own errors at the methodological
level. This is consistent with Type G: the method cannot fully step outside itself to audit itself.
