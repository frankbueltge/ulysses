# Provenance — texts/

## melanctha.txt

"Melanctha. Each One As She May" — the middle novella of Gertrude Stein, *Three Lives*
(written 1905–06, first published by The Grafton Press, New York, 1909). Public domain
(published before 1930).

Source: Project Gutenberg eBook #15408, *Three Lives*,
<https://www.gutenberg.org/ebooks/15408> (plain-text file
<https://www.gutenberg.org/cache/epub/15408/pg15408.txt>), retrieved 2026-07-16 via
web research (full-text extraction; direct download is blocked in this environment).
The file here is the untouched section from the heading "MELANCTHA" to "FINIS"
(the end of the novella, before "The Gentle Lena") — 50,120 words. No edits beyond
that excision and trailing-whitespace trim. The Project Gutenberg header/footer and
license text belong to the full file at the URL above, not to this excerpt; the text
itself is public domain.

Chosen because it is the long prose work in which Stein's repetition-as-insistence
is *practised* (she lectures about it in "Portraits and Repetition", 1935, which is
not yet public domain and is therefore only briefly quoted, with citation, in the
work itself).
