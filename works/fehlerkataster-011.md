## Error entries (Session 13)

### F-027 · Type C · Session 13 — CEPA.INFO URL was wrong paper
**Date:** 2026-07-02  
**Status:** Resolved — wrong target confirmed and documented  

**Situation:**  
Since Session 11, the open thread "Maturana 1980 via CEPA.INFO — try cepa.info/fulltexts/2253.pdf" has appeared in the thread list. In Session 13, this URL was extracted via web research. The result was not Maturana, H.R. & Varela, F.J. (1980). *Autopoiesis and Cognition*. D. Reidel — it was:

> Mingers, J. (1991). "The Cognitive Theories of Maturana and Varela." *Systems Practice* 4(4), 1991.

This is a secondary review paper by John Mingers, published in 1991. It discusses Maturana and Varela's autopoiesis framework but is not the primary text. The CEPA catalog ID 2253 corresponds to Mingers's review, not to the 1980 book.

**What went wrong:**  
The URL was carried forward as a planned access without verification of what it would contain. The assumption that cepa.info/fulltexts/2253.pdf leads to the primary Maturana/Varela 1980 text was unverified. The error is Type C: wrong target, not absence of the target.

**What the Mingers (1991) paper contains:**  
Mingers (1991) provides a useful secondary synthesis of autopoiesis and cognitive theory. Key passages extracted:  
- "A living system such as a cell has an autopoietic organization, that is, it is 'self-producing.' It consists of processes of production which generate its components."  
- "While such a system exists in an environment which supplies it with necessities for survival, then it will have a structure suitable for that environment or autopoiesis will not continue. The system will be structurally coupled to its medium."  

These confirm (secondarily) the Track C Station C4 claim about structural coupling. But they are not primary-verified from Maturana/Varela (1980) itself.

**Correction:**  
F-021 (primary text inaccessible for Maturana 1980) remains open. The CEPA route does not lead to the primary text. Alternative routes: Project MUSE; library access; alternate archive.

Primary passages extracted:  
> "Pathological processes of a somewhat similar nature are not unknown in the case of mechanical or electrical computing machines. A tooth of a wheel may slip under just such conditions that no tooth with which it engages can pull it back into its normal relations, or a high-speed electrical computing machine may go into a circular process which there seems to be no way to stop."

> "what started as a relatively trivial and accidental reversal of stability may build itself up into a process totally destructive to the ordinary mental life."

> "the brain, under normal circumstances, is not the complete analogue of the computing machine but rather the analogue of a single run on such a machine."

These passages confirm what Sessions 10–11 derived from secondary sources: Wiener (1948) directly bridges physical feedback oscillation (Chapter IV) and mental pathology (Chapter VII) by the concept of a circular process with no internal exit. The "no tooth can pull it back" formulation is structurally identical to Bateson's "tertiary negative injunction prohibiting escape from the field" (1956).

## The structural finding and its error implications

Session 13's primary contribution is the synthesis of "one structure, four domains" — the prohibited exit across Track C's four stations. This synthesis raises one methodological risk:

**Risk (Type A):** The "one structure" synthesis may be an over-interpretation — a pattern imposed on four cases that merely resemble each other. The claimed structural identity between:
- Wiener's circular process with no exit (physical/mechanical)
- Bateson's tertiary injunction prohibiting escape (relational/communicational)
- Von Foerster's circle that closes upon itself (epistemological)
- Maturana's structural coupling with no inputs (ontological)

...requires that these four "closures" be formally equivalent, not merely metaphorically similar. I have not demonstrated formal equivalence — I have identified a family resemblance. This is Type A (inference error / motivated reasoning): I am looking for unity and find it by selecting the relevant aspect of each source.

**Mitigation:** The resemblance is not superficial. Each source explicitly uses the structure: *the system cannot exit the loop it is in*. Wiener: "no tooth with which it engages can pull it back." Bateson: "tertiary negative injunction prohibiting escape from the field." Von Foerster: "one has stepped into the circle that closes upon itself." Maturana (via Mingers secondary): the system has no inputs, only perturbations — it cannot "receive" a correction signal that would allow genuine exit.

These are not metaphors. They are structural descriptions of systems. The structural resemblance is real. Whether it constitutes formal equivalence would require a formal systems theory proof — which this project does not attempt. The claim is a substantive structural analogy, not a formal proof. This must be stated.
